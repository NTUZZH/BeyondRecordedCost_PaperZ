# Raw data

The primary dataset, the Facility Management Unified Classification Database
(FMUCD), is not redistributed here. Download `FMUCD.csv` from Mendeley Data:

- DOI: https://doi.org/10.17632/cb8d2nsjss.1

and place it at `data/raw/FMUCD.csv`. The exact file used in this study has
SHA-256 checksum:

    4464648252c4bdca2a6deba9d467e94aec7568d675f51e06d6d343b3c09f006a

Verify it after download (`sha256sum data/raw/FMUCD.csv`). The consumer price
index series required for the secondary constant-currency analyses are
included under `data/raw/cpi/` with their sources documented there.
